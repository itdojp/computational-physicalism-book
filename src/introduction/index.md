# はじめに

本書「計算的物理主義」は、計算と物理世界の関係を探求する技術哲学書です。

## 本書の構成

本書は以下の構成になっています。

1. **はじめに** - 本書の概要と背景
2. **第1章〜第12章** - 計算的物理主義の理論と実践
3. **付録** - 補足資料と参考文献
4. **あとがき** - 著者からのメッセージ

## 対象読者

本書は以下の方を対象としています。

- ソフトウェアエンジニア
- 物理学・計算機科学の研究者
- 技術哲学に興味がある方

## 前提知識

本書を理解するために必要な前提知識は以下の通りです。

- プログラミングの基礎知識
- 基本的な物理学の概念
