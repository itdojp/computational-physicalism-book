# はじめに

本書「計算論的物理主義」は、計算と物理世界の関係を探求する技術哲学書です。

## この章の役割

この章では、読み進める前に次の3点を把握します。

- 本書がどの順序で論点を積み上げるか
- 自分の関心に応じて、どの章から読み始めるか
- 議論が重く感じたときに、どの付録へ戻ればよいか

## どこから読むか

- 本書全体の立場を最初に押さえたい場合は、[第1章：計算論的物理主義の基礎](../chapters/chapter01/) から読み始めてください。
- 計算可能性や計算量の整理を優先したい場合は、[第2章：計算可能性理論による厳密な証明](../chapters/chapter02/) を先に読んでも構いません。
- 既存AI研究との比較から入りたい場合は、[第3章：既存AI研究との比較分析](../chapters/chapter03/) に進み、必要に応じて第1章・第2章へ戻ってください。
- 実装や設計への落とし込みを重視する場合は、[第4章：実装への道筋](../chapters/chapter04/) を軸に読み、第2章と往復してください。
- 倫理・社会的インパクトを先に確認したい場合は、[第5章：倫理的・社会的含意](../chapters/chapter05/) を先に読み、その後で第1章〜第3章に戻ると整理しやすくなります。

## 先に引く付録

- [付録A：用語集](../appendices/appendix01/)  
  Physical CTT、PAC学習、創発などの用語が曖昧な場合は、先に確認してください。
- [付録B：参考文献一覧](../appendices/appendix02/)  
  特定の主張の根拠文献をすぐ追いたい場合は、本文と並行して参照してください。

## 読み進め方の目安

1. [第1章](../chapters/chapter01/) で立場と争点を把握する
2. [第2章](../chapters/chapter02/) と [第3章](../chapters/chapter03/) で理論的根拠と既存研究との差分を確認する
3. [第4章](../chapters/chapter04/) と [第5章](../chapters/chapter05/) で実装・運用・社会的含意へ接続する
