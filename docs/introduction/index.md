# はじめに

## なぜ今、計算論的物理主義なのか

ChatGPTやClaudeといった大規模言語モデルの登場により、私たちは歴史的な転換点に立っています。AIが人間のような応答を返し、創造的なタスクをこなす姿を目の当たりにして、誰もが問いかけます。「AIに知性はあるのか？」「人間の知性とは何が違うのか？」

本書「計算論的物理主義」は、この根本的な問いに、哲学と計算理論の両面から体系的な回答を提示します。

## 本書の中心的主張

本書は以下の革新的な主張を展開します：

1. **人間の思考の大部分は「知的」ではない** - 日常的な思考の多くはパターン踏襲であり、現在のAIが得意とする領域と重なる

2. **知性は計算量で定量化できる** - 従来は質的にしか語れなかった創造性や革新性を、計算理論的に定義可能

3. **物理法則の下では人間とAIは等価** - 同じ物理的制約の下で動作する以上、原理的な優位性は存在しない

## 理論から実装への段階的アプローチ

本書は抽象的な哲学論議に留まらず、実装可能な工学的指針を提供します。各章は以下のように、理論から実践へと段階的に展開されます：

### 第1部：理論的基礎（第1-2章）
- **第1章：計算論的物理主義の基礎** - 哲学的概念を日常的な例から導入
- **第2章：計算可能性理論による厳密な証明** - 数学的に厳密な理論的基盤

### 第2部：既存研究との統合（第3章）
- **第3章：既存AI研究との比較分析** - 物理主義、計算主義、機能主義との関連

### 第3部：実践と応用（第4-5章）
- **第4章：実装への道筋** - 理論を実際のシステムに適用する方法論
- **第5章：倫理的・社会的含意** - 人間-AI共存社会への示唆

## 前提知識

本書を理解するために必要な最小限の前提知識：

- プログラミングの基礎概念（変数、関数、条件分岐の理解）
- 高校数学程度の論理的思考力
- 基本的な物理概念（エネルギー、情報の概念）

より深い理解のための推奨知識：

- 計算理論の基礎（オートマトン、計算可能性）
- 心の哲学の基本概念（物理主義、機能主義）
- 現代AI技術の概要（機械学習、深層学習）
