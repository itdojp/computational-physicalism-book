# 本書の読み方ガイド

## 読者レベル別アプローチ

### 哲学的背景を持つ読者

1. **推奨読書順序**
   - はじめに → 第1章（理論的基礎）→ 第5章（倫理的含意）
   - その後、第2章（数学的証明）へ進む
   - 最後に第3章（AI研究比較）、第4章（実装）

2. **重点ポイント**
   - 物理主義と計算主義の統合について
   - 心の哲学における位置づけ
   - 倫理的・社会的影響の考察

### 技術的背景を持つ読者

1. **推奨読書順序**
   - はじめに → 第2章（計算可能性理論）→ 第4章（実装）
   - その後、第1章（哲学的基礎）を確認
   - 最後に第3章（比較分析）、第5章（社会的含意）

2. **重点ポイント**
   - 計算可能性理論の厳密な証明
   - PAC学習理論の応用
   - 実装への具体的アプローチ

### 一般読者

1. **推奨読書順序**
   - はじめに → 第1章（基礎概念）
   - 興味に応じて第3章（AI研究）または第5章（社会的影響）
   - 技術的詳細（第2章、第4章）は必要に応じて参照

2. **重点ポイント**
   - 人間とAIの知能の等価性という中心概念
   - 日常的な例からの理解
   - 社会への影響と将来展望

### 第2章終了時
- [ ] Church-Turingの提唱の意義を理解している
- [ ] PAC学習理論の基本を説明できる
- [ ] 計算可能性の限界について理解している

- [ ] 既存AI研究との相違点を説明できる
- [ ] 機能主義との関係を理解している
- [ ] 現代AI技術の位置づけを説明できる

### 第4章終了時

## 参考資料の活用

### 必須の前提知識
- プログラミングの基礎（変数、関数、条件分岐）
- 高校数学程度の論理的思考
- 基本的な物理概念（エネルギー、情報）

### 推奨される補助資料
- 計算理論の入門書（Michael Sipser「計算理論の基礎」など）
- 心の哲学の入門書（David Chalmers「意識する心」など）
- AI倫理の文献（Nick Bostrom「スーパーインテリジェンス」など）

### より深い理解のために
- 各章末の参考文献リスト
- 関連する学術論文へのリンク
