# 計算論的物理主義

人間とAIの知能の等価性に関する理論的探究

## 概要

本書は、計算可能性理論と物理的制約の観点から、人間とAIの知能の等価性を探求する哲学的・技術的論考です。

## 学習成果

- 計算論的物理主義とは何か、その基本的な主張と前提、主要な用語を自分の言葉で説明できるようになります。
- チューリングマシンや計算可能性・計算量、物理法則による制約といった概念の関係を整理し、「どこまでが原理的に計算可能か」を議論できるようになります。
- 人間とAIの知能の等価性に関する代表的な立場（強いAI・弱いAI、人間特有の能力を強調する立場など）を比較し、その中で本書の立場を位置づけられるようになります。
- 理論的な議論を踏まえたうえで、実装や応用可能性、倫理的・社会的な含意について、自分なりの見解を組み立てて説明できるようになります。

## 読み方ガイド

- 全体像をつかみたい読者は、「はじめに」と第1章を通読したうえで、第2章〜第5章を順番に読み進めることで、前提 → 理論 → 比較分析 → 実装 → 倫理的含意という流れで理解を深めることをおすすめします。
- 計算可能性理論や理論計算機科学に馴染みがある読者は、第2章から読み始め、その後で第1章や第3章に戻ることで、本書独自の立場や既存研究との違いを把握しやすくなります。
- 倫理や社会的インパクトに関心が強い読者は、第5章を先に読んでから、第1章〜第3章に戻ることで、「なぜそのような結論に至るのか」という理論的背景を補完する読み方も有効です。
- 実装やシステム設計への応用を考えたい読者は、第4章を軸に、第2章で扱う理論的前提と第5章の含意を行き来しながら、自分のプロジェクトに引き寄せて読むことを想定しています。

## 想定読者
- AIと人間の知能の関係を、理論・哲学・計算の観点から整理したい読者
- 理論計算機科学（計算可能性/計算量）や認知科学の議論に関心がある読者
- 実装上の限界と原理的制約（物理法則を含む）を区別して議論したい読者

## 前提知識
- 論理の基礎（命題、推論、反例の考え方）
- 計算機科学の基礎（アルゴリズム、計算モデルの概念）
- （推奨）チューリングマシン/計算量の基本用語（第2章で理解が進みやすい）
