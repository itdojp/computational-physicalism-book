# 計算論的物理主義

人間とAIの知能の等価性に関する理論的探究

## 概要

本書は、計算可能性理論と物理的制約の観点から、人間とAIの知能の等価性を探求する哲学的・技術的論考です。

## 内容

- **はじめに**: 本書の背景と目的
- **第1章**: 計算論的物理主義の基礎
- **第2章**: 計算可能性理論による厳密な証明
- **第3章**: 既存AI研究との比較分析
- **第4章**: 実装への道筋
- **付録**: 数学的補足と参考文献

## ビルド方法

```bash
# 依存関係のインストール
```
